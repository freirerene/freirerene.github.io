Machine Learning

  • Linear Regression on Price of Rent : We continue our little project of analysing the price of apartaments for rent in São Paulo. In this instance we apply multivariate linear regression.

Cleaning Jobs

  • Cleaning Exit Surveys : Here we clean two very messy surveys on employee termination and we want, at first, to see if there's a relation between age/length of service and resignation due to dissatisfaction. In the end we clean the survey and make a useful dataframe for further HR use.
  • Cleaning Apartaments for Rent : We clean the data we collected from the web, containing information about apartaments for rent in São Paulo. We also use K-Nearest Neighbors to determine the missing values for a column.

SQL

  • The CIA Factbook : This project is meant to be a simple exploration of a database containing some informations about the countries around the world. We calculate the percentile of population growth, death rates and other things.
  • SQL Business Questions : In this notebook we answer a couple of business questions for a fictional music store using SQL to access its database.

Finance / Risk

  • Hurst I and Hurst II : In here we calculate the Hurst exponent, that estimates if a Time Series is stationary or not (we also compare to the ADF test).
  • Monte Carlo Drawdown : Here we implement a Monte Carlo to estimate the risk of a strategy we developed somewhere else. We calculate the mean and the maximum drawdown of the strategy.
  • Risk of Ruin in UNO : We applied the same Monte Carlo method to a game of UNO -- also, we tweaked it to establish how much it would change if a player had some edge, etc.

Fetching Data

  • Collect Data on Apartaments for Rent : We collect some data using selenium. Because using requests together with beautifulsoup directly was resulting in us being blocked by captcha, we solved this problem using selenium, with a new browser for every fetch.
  • Financial News : We use requests together with beautifulsoup to construct a simple bot that emails you the financial news that are expected to be of high impact.

Visualization

  • Visualizing the Evolution of COVID per state : As the title suggests we use tableau to visualize the growth of the death per 100k inhabitans from February 2020 to October 2020 for each Brazilian state.
